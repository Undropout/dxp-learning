Edit a Step

Use the following procedure to edit an existing step:

  1. In the directory carousel, click the project that contains the step that you want to update.
  2. In the directory, click the step that you want to update.
  3. Click Edit in the Design sub-tab.
  4. Click the Settings icon in the right pane of the Design sub-tab:

  1. Update the following fields in the right pane as needed:
    • Enter a new name into the Step Name field. The name must be between 5 and 35 characters in length and must contain only letters, numbers, and other approved characters.

    Note: Blank spaces are not allowed.

    • Select a resource to flush from the Flush on Modify drop-down menu.
    • Select a template for the step from the Template drop-down menu.
  2. To update the data profile for the step, click Data Profile in the right pane of the Design sub-tab:

    3. Note: You can drag objects or properties from the right pane and drop them into the data profile section as needed.

  3. To edit field settings, click an object or property in the Data Profile section, and the Field Setting section displays:

  4. Select the following options as needed:
    • Default – Select this checkbox, and then select Static to apply a static value or select Dynamic to return a dynamic default value. Click New Rule to create a precondition. For example, if the value is X, return a certain value. For more information on creating a static value, see Create a List.
    • Mandatory – Select this checkbox to make this object mandatory. Click New Rule to create a precondition. For example, if the value is X, then make the field mandatory.
    • Read-Only – Select this checkbox, to make this object read-only. Click New Rule to create a precondition. For example, if the value is X, then make the field read-only.
    • Disabled – Select this checkbox to disable the object. Click New Rule to create a precondition. For example, if the value is X, then disable the field.
    • Hidden – Select this checkbox to hide the object. Click New Rule to create a precondition. For example, if the value is X, then hide the field.
  5. Edit the following field settings as needed: 
    • To make this object searchable, select the Searchable checkbox.
    • To change the group to which the object belongs, enter the group number into the Preferred Group field.
    • To reorder the object, enter the order number into the Preferred Order field.
    • To add a UI hint, select it from the UI Hints drop-down list. For more information, see Create a UI Hint.
    • Click Save.
  6. To add, edit, or delete a step create link, click Integrations & Rules in the right pane of the Design sub-tab, and then click the Create box:

  7. Edit the step create link as needed:
    • Select the Precondition checkbox to add an existing precondition. Click New Rule to add a rule to the precondition.
    • Click Add Service to add a service. In the Name field, edit the existing service name. Select the Precondition checkbox to add an existing precondition. Click New Rule to add a rule to the precondition.
    • Select an existing service from the Service Type drop-down menu.
    • Click the X on the Create box to delete the step create link.
  8. To add, edit, or delete a step edit link, click the Edit box in the Integrations & Rules area:

  9. Edit the step edit link as needed:
    • Select the Precondition checkbox to add an existing precondition. Click New Rule to add a rule to the precondition.
    • Click Add Service to add a service. Enter a name into the Name field. Select the Precondition checkbox to add an existing precondition. Click New Rule to add a rule to the precondition.
    • Select a service type from Service Type drop-down menu.
    • Click the X on the Edit box to delete the step edit link.
  10. Click Save.
