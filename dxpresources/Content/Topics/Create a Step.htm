Create a Step

Use the following procedure to create a new step:

  1. In the directory carousel, click the project that contains the application and business feature to which you want to add a step.
  2. In the directory, click the application, and then the business feature to which you want to add the step.
  3. Hover your cursor over the business feature and click the Menu icon:

  1. Click the Add icon, and the Add Step window displays:

  1. Enter a name for the new step into the Step field. The name must be between 5 and 35 characters in length and must contain only letters, numbers, and other approved characters.

Note: Blank spaces are not allowed.

  1. If desired, select a template from the Template drop-down menu:
    • CTA Template – Choose this option to add a CTA (call to action) template. For more information, see Manage CTA Templates.
    • Simple Form Template – Choose this option to add a simple form template, which is mostly used when you are returning only data. For more information, see Manage Simple Form Templates.
    • There may be other existing custom templates that you can select.

    Note: When a template is selected, the name of the template displays on the Step screen.

  1. Click Save, and the Step screen displays.
  2. Click Edit, and the Edit Step screen displays:

  1. Click the Data Profile sub-tab.
  2. In the ADM Library box, enter the name of the object or property that you want to add to the create link.

Note: If the ADM Library box is empty, you must create at least one object or property. For more information, refer to the Application Data Model (ADM) section.

  1. Drag the desired object or property from the ADM Library box and drop it into the Data Profile section of the Step window.
  2. Add create and edit links to the step as described in the Edit a Step section.
  3. If desired, customize the new step as described in the following sections:
    • Add a Step Link Precondition
    • Add a Step Service
    • Add User Notes
  4. Click Save.
