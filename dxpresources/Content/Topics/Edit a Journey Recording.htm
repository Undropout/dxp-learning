Edit a Journey Recording

After you record a journey, you can edit the following elements:

  • Request/response message
  • Recording name, description, playback mode, or message

Edit the Request/Response Message

Complete these steps to edit the request/response message:

  1. Clone the recording that you want to edit.
  2. Click the cloned copy in the list of recordings.
  3. Click the message that you want to edit, and the Request/Response form displays:

  4. Click Edit, and the recording details display in the right pane:

  5. Complete these steps as needed to edit the recording details:
    • Enter the desired information in the PathThe API endpoint path and/or HeadersRequest headers of the message fields.
    • To delete a header, click the X icon.
    • To add a header, click the + icon, and then enter a key and value for the header.
  6. Click Save.

Edit the Recording Name, Description, Playback Mode, or Message

Complete these steps to edit the recording name, description, playback mode, or message:

  1. Clone the recording that you want to edit.
  2. Click the cloned copy in the list of recordings.
  3. Click Edit on the recording details screen:

    4. The following dialog box displays:

  4. Complete these steps on the Recording tab, as needed:
    • Enter the desired information into the Name and Description fields to edit the name and description.
    • Click the drop-down arrow and select a playback mode from the list.
  5. To edit a message, click the Messages tab, and the following dialog box displays:

    6. Note: Click the X icon next to a message to delete it.

  6. Click Save.
