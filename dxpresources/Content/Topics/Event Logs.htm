Event Logs

Journey Creator keeps a running log of all events that occur in the session. This data can be a useful tool to troubleshoot issues that you encounter.

To view the event log, click the Admin icon, and then click the Event Log sub-tab:

The Event Log screen contains the following information for the current selected project, when available:

  • When the event was created
  • Who created the event (v4.2)
  • The event type
  • The entity type
  • The entity name
  • The associated workstream (v4.2)

Note: You can use the Search field to search for specific events.
