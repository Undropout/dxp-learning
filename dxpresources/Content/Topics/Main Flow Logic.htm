Main Flow Logic

While configuring a main flow, you can add logical decisions to the flow. These decisions can be configured in many different ways to achieve your intended results.

Note: Refer to the Main Flows section for more information about creating a main workstream flow.

Use the following procedure to add decision logic to a main flow:

  1. Navigate to the Design sub-tab.
  2. Select the desired application in the directory:

  1. Click the Edit icon in the design area, and the workstream design area displays:

  1. Click Logic in the design area.
  2. In the Logic section, drag and drop the Decision feature into the work area to begin creating logic for the flow:

Note: Click Add Decision to add another decision to the decision block, or click the Trash icon to delete a decision from the decision block.

  1. Click the Edit icon for the first decision, and the New Decision Expression window displays:

  1. Enter a unique name for the expression into the Name field.
  2. Use the options available in the sub-tabs to drag, drop, and connect puzzle pieces to create the desired logical expression in the work area:

Note: You can click Switch to Code Editor to create an expression using the code editor, but you will not be able to return to the puzzle editor after the code editor is selected.

  1. Click OK to save the expression, and then continue to add decisions as needed.

Note: Refer to the Add a Business Rule section for more information.

  1. Use the orange squares provided in each business feature and decision bubble to connect them as desired (drag and drop) for the intended UI flow:

  1. Click Save to save the flow.
