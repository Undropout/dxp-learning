Define Privacy Settings

You can add privacy settings to arrays and properties.

Complete these steps to define privacy settings:

  1. In the directory carousel, click the project for which you want to define privacy settings.
  2. In the directory, click the application for which you want to define privacy settings.
  3. Click the Data Editor tab.
  4. Click the ADM sub-tab.
  5. Click one of the following:
    • The Array link to add privacy settings to an array
    • The Property link to add privacy settings to a property
  6. In the right pane, click Privacy, and the Privacy area displays:

  7. To display all of the user entries, click Public.
  8. To conceal some or all of the user entries, click Confidential. Click the Mask drop-down arrow and select how to conceal the entry.
  9. Click Save.
